# K8s(kind), Docker Compose, GCloud
An example setup script and manifests demonstrating docker-compose, kind, and glcoud.
See `.brev/setup.sh` for details.

## Run

`gcloud init`

`docker-compose up`

`kubectl apply -f k8s-pod-manifest.yml`
